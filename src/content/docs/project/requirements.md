## 需求优先级说明

采用 MoSCoW 方法标记优先级：

| 标记 | 含义 | MVP 阶段 |
|------|------|----------|
| M (Must) | 必须实现，核心功能 | 包含 |
| S (Should) | 应该实现，重要功能 | 部分包含 |
| C (Could) | 可以实现，增强功能 | 不包含 |
| W (Won't) | 本期不做 | 不包含 |

## 功能需求清单

### 文档站基础 [M]

| ID | 需求描述 | 验收标准 |
|----|----------|----------|
| REQ-001 | Astro + Starlight 项目初始化 | 本地 npm run dev 可正常启动 |
| REQ-002 | 自定义主题配色 | 配色方案与品牌一致，支持深色模式 |
| REQ-003 | 首页设计 | 包含项目介绍、快速导航 |
| REQ-004 | 文档分类目录结构 | 支持 3 大分类 |

### 内容展示 [M]

| ID | 需求描述 | 验收标准 |
|----|----------|----------|
| REQ-005 | Markdown 渲染 | 支持 GFM 语法 |
| REQ-006 | 代码高亮 | 支持 20+ 编程语言 |
| REQ-007 | 内置搜索 | 支持中文搜索 |
| REQ-008 | 响应式布局 | 多端自适应 |

### 可视化 [M]

| ID | 需求描述 | 验收标准 |
|----|----------|----------|
| REQ-009 | Mermaid 图表集成 | 支持 flowchart、mindmap |
| REQ-010 | 思维导图渲染 | mindmap 语法可正确渲染 |
| REQ-011 | 速查表样式 | 提供专用 CSS 类 |

### AI 生成管道 [M]

| ID | 需求描述 | 验收标准 |
|----|----------|----------|
| REQ-012 | learning-master Skill | 主控 Skill 可接收主题输入 |
| REQ-013 | topic-analyzer Skill | 输出标准化 JSON |
| REQ-014 | outline-planner Skill | 生成三阶段框架大纲 |
| REQ-015 | content-writer Skill | 分段生成内容 |

| REQ-017 | quality-checker Skill | 检查通过率 >= 80% |
| REQ-018 | 一键生成命令 | /learn 命令可触发完整流程 |

### 学习框架支持 [S]

| ID | 需求描述 | 验收标准 |
|----|----------|----------|
| REQ-023 | 概览阶段内容 | 包含定义、问题、场景、前置知识 |
