# 快速开始指南

欢迎使用 StudyBuddy！本指南将帮助你快速了解如何使用这个知识学习平台。

## 三阶段学习法

每个主题都遵循「三阶段学习框架」：

### 概览（5分钟）

快速了解主题的核心概念：
- 一句话定义
- 核心问题
- 适用场景
- 前置知识
- 思维导图

### 详解（60分钟）

深入学习各个知识点：
- 分章节讲解
- 每个概念包含「是什么 - 为什么 - 怎么用」
- 代码示例和图表辅助

### 实战（25分钟）

通过练习巩固所学：
- 初级难度：基础应用
- 中级难度：综合运用
- 高级难度：复杂场景

## 使用 AI 生成文档

在 Qoder 中，你可以使用以下 Skill 命令自动生成学习文档：

```
/learning-master TypeScript
```

AI 将自动分析主题，生成结构化的学习内容。

## 浏览方式

### 按分类浏览

- [**工具类**](/tools/) - 各类软件工具的使用方法
- [**领域类**](/domains/) - 技术领域的知识体系
- [**方法论**](/methods/) - 学习方法和思维框架

### 使用搜索

点击右上角的搜索图标，输入关键词快速找到相关内容。

### 知识图谱

使用页面底部的图谱功能，探索文档之间的关联关系。

## 下一步

开始浏览你感兴趣的主题，或者尝试使用 `/learn` 命令生成你想学习的内容！
